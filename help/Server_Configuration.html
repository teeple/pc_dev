  5. b)go to /var/lib/tomcat6/webapps/prdcat/js/impl

  7. d)open the file ‘global.js’

  1. c)open the file ‘restsupport.js’

          URL_PREFIX: “http://[ProductCatalog_SERVER_IP]”,

    save and close the ‘restsupport.js’ file
