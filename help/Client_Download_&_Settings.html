Client Setting

 

PRE-REquirement

  1. -Adobe AIR installed machine.

  2. -If not installed Adobe Air Runtime follow the instructions

  3. a)Open Browser and Goto http://www.adobe.com/download

  1. b)Get the Adobe AIR latest version

Client SETTING

c) Download Adobe AIR and Install

Client Installation Process:

a) Click UAngelProductCatalog.air to download product catalog client download

  1. b)Open UAngelProductCatalog.air file and Install

    Note: Don’t check the Run after Install.  If checked, then you should quit the program after install with Alt + F4 and follow below procedures.

c) Goto Program Files/Product Catalog directory

d) Open assets/config.xml for edit.

  1. e)Change the IP

    <HostAddress>http://[SERVER_IP]:3000</HostAddress>

   <WebHostAddress>http://[SERVER_IP]</WebHostAddress>

   Note: HostAddress’s Port is 3000. WebHostAddress doesn’t have port number.

  1. f)Run the Product Catalog
